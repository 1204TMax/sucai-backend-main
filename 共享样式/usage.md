# 共享样式使用说明

本目录存放工作流后台项目的共享样式和设计规范。

## 文件说明

| 文件 | 说明 |
| --- | --- |
| `backend.css` | 后台页面共享 CSS，包含设计 token、布局、按钮、表格、弹窗、表单等基础样式 |
| `design-system.md` | 后台设计系统说明 |

## 页面引用方式

所有后台页面统一引用：

```html
<link rel="stylesheet" href="共享样式/backend.css">
```

如果页面位于子目录，需要根据实际相对路径调整。

## 页面布局规范

- `body padding` 使用 `8px`。
- 页面大模块之间 `gap` 使用 `8px`。
- 控件高度使用 `32px`。
- 基础字号使用 `13px`。
- 卡片和面板圆角不超过 `8px`。
- 后台页面以表格、筛选、弹窗和表单为主，不做营销式首屏。

## 命名规则

后台共享样式使用通用但受控的类名，例如：

- `.app-shell`
- `.sidebar`
- `.panel`
- `.toolbar`
- `.table`
- `.modal`
- `.tag`

当前后台为独立项目，不与主站页面样式混用。
